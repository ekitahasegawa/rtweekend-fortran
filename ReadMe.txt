========================================================================
    Fortran Console Application : "rtweekend-fortran" Project Overview
========================================================================

This is an implementation of Peter Shirley's "Ray Tracing in One Weekend"
in modern Fortran.

The ebooks and source code can be found for free online here: 
https://raytracing.github.io/

Note that I've included the Visual Studio project and solution files
for convenience, but as of Visual Studio 2022 version 17.3.6
there are issues with creating and loading fortran projects.
